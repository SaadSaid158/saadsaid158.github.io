About Saad Said

Projects

  • Fusée Gelée PoC — Go-based implementation of CVE-2018-6242, a coldboot BootROM exploit targeting NVIDIA Tegra X1 chips in Nintendo Switch units. Features TUI, payload management, SHA256 verification, multi-device support, and JSON config. Cross-platform.

  • Chess Engine — A compact chess engine in Go with bitboard board representation, iterative deepening alpha-beta search, and perft validation. Two front ends: a local browser UI and UCI mode for standard chess GUIs.
  • C2 Framework — A modular red team command-and-control framework. Cross-platform C implants (Windows + Linux) with a Go teamserver, multi-operator CLI, OTA update system, and a production build pipeline.
